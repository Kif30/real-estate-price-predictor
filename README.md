
# 🏠 End-to-End House Price Prediction

## 📌 Project Overview
This project focuses on predicting residential house prices using supervised machine learning techniques.
The objective is to analyse housing data, perform preprocessing and feature engineering, and build regression models to estimate property prices accurately.

## 🎯 Problem Statement
Real estate price estimation is important for buyers, sellers, and investors.
This project aims to build a predictive model that can estimate house prices based on multiple input features such as property size, number of rooms, and other attributes.

## 📊 Dataset
- Housing dataset with numerical and categorical features
- Includes variables like area, bedrooms, bathrooms, etc.
- Data preprocessing performed to handle missing values

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- Google Colab

## 🧠 Models Used
- Linear Regression
- K-Nearest Neighbors Regressor

## 🔎 Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Model Comparison

## 📈 Evaluation Metrics
- MAE
- MSE
- R2 Score
